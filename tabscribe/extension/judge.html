Judge Mode

Quick start: Select text → Save (Alt+S) → Open side panel → Summarize/Rewrite/Translate → Draft Report → Export.

Features

  • Offline by default (Gemini Nano), Hybrid optional
  • Multimodal: image explain and audio notes
  • Evidence snapshots and citations
  • Markdown/.docx export
